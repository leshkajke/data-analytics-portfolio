# SQL Patterns
# SQL Patterns for Data Analyst

В этом разделе собраны типовые SQL-паттерны,
которые используются в ежедневной работе аналитика данных.

## Что внутри
- агрегации и группировки
- join-ы
- оконные функции
- когортный анализ и retention
- воронки
- работа с датами

## Типы данных
Целые числа
INT
INTEGER
BIGINT
SMALLINT

Дробные числа
DECIMAL(10,2)
NUMERIC(10,2)
FLOAT
REAL

Тесктовый
VARCHAR(255)
TEXT
CHAR(10)

Дата/время
DATE - дата
TIMESTAMP - дата + время
TIMESTAMP WITH TIME ZONE

Логический тип
BOOLEAN

NULL (отсутствие значения)
price IS NULL
price IS NOT NULL



## Как использовать
Каждый файл:
- описывает бизнес-задачу
- показывает структуру запроса
- содержит комментарии по логике

Паттерны можно адаптировать под реальные данные
(PostgreSQL / MySQL / BigQuery).
